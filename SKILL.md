# Feishu Permission Tool

Single tool `feishu_perm` for managing file/document permissions.

## Actions
- list - List collaborators
- add - Add collaborator (member types: email, openid, userid, unionid, openchat, opendepartmentid)
- remove - Remove collaborator

## Permission Levels
- view - View only
- edit - Can edit
- full_access - Can manage permissions

## Token Types
doc, docx, sheet, bitable, folder, file, wiki, mindnote

## Permissions
Required: drive:permission
